UmbraCare.StarRating is free and licensed under MIT license.

The trial version of the Dedicated Media Folder package is also installed with a Star Rating plugin.

If you want, you can uninstall the Dedicated Media Folder package at any time and continue using Star Rating.

All you need to do during package uninstallation is to check the option: 
    Force uninstall, even if there are dependencies on it.

You can also uninstall this package in Package Manager Console using below command:
    Uninstall-Package UmbraCare.DedicatedMediaFolder -Force

However, please consider checking out the Dedicated Media Folder before you uninstall it because we are sure you will like it.

More about the Dedicated Media Folder package you can find at: https://umbracare.net/products/dedicated-media-folder
